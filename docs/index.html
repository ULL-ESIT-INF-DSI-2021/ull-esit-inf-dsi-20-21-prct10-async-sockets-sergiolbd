Colecciones genéricas (https://howtodoinjava.com/typescript/maps/)

  • Map<K, V>: Permite definir un mapa cuyas claves son de tipo K y sus valores de tipo V.

    • Crear un MAP --> let myMap = new Map([[key, value], [key2, valor2]])

    • Acceder a la clave --> myMap.keys()

    • Acceder al valor --> myMap.values()

    • Añadir elemento --> myMap.set('x', 'y')

    • Encontrar un elemento dada su clave --> myMap.get('x')

    • Encontrar la existencia de un elemento --> myMap.has('x')

    • Eliminar --> myMap.delete('x')

    • Limpiar --> myMap.clear()

    • Tamaño --> myMap.size

    • Iterar sobre el mapa: 

        let nameAgeMapping = new Map();

  nameAgeMapping.set("Lokesh", 37);
  nameAgeMapping.set("Raj", 35);
  nameAgeMapping.set("John", 40);
  
  //Iterate over map keys
  for (let key of nameAgeMapping.keys()) {
      console.log(key);                   //Lokesh Raj John
  }
  
  //Iterate over map values
  for (let value of nameAgeMapping.values()) {
      console.log(value);                 //37 35 40
  }
  
  //Iterate over map entries (RECORRER ELEMENTO A ELEMENTO)
  for (let entry of nameAgeMapping.entries()) {
      console.log(entry[0], entry[1]);    //"Lokesh" 37 "Raj" 35 "John" 40
  }
  
  //Using object destructuring
  for (let [key, value] of nameAgeMapping) {
      console.log(key, value);            //"Lokesh" 37 "Raj" 35 "John" 40
  }

  • ReadonlyMap<K, V>: Permite definir un Map que no puede modificarse.

  • Set: Permite definir un conjunto cuyos elementos son de tipo T.
    • Crear un set --> let myset = new Set()
    • Añadir elemento --> set.add()
    • Comprobar la existencia de un elemento --> set.has()
    • Eliminar --> set.delete(x)
    • Limpiar --> set.clear()
    • Tamaño --> set.size
    • Iterar sobre un set:
      • for (let currentNumber of set) { currentNumber contiene un número y no un índice}
      • set.forEach( function(value) {console.log(value)})
  • ReadonlySet: Permite definir un conjunto no modificable.
